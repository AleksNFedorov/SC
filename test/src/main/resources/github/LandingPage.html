Главная страница

При переходе на сайт https://www.github.com открывается страница проекта, на которой должно быть (сверху вниз)

Панель с формой поиска и ссылками

На панели должны находиться картинка с логотипом проекта, поле для воода строки поиска с подсказкой Search GitHub, 4 основные ссылки
Текст ссылки Ссылка Отображается Активный
Explore ${github.url.prefix}/explore Yes Yes
Features ${github.url.prefix}/features Yes Yes
Enterprise https://enterprise.github.com/ Yes Yes
Blog ${github.url.prefix}/blog Yes Yes
И кнопка (Sign in) для входа
На панели находится слоган крупным тесктом (h1) “Build software better, together.”,
далее текст “Powerful collaboration, code review, and code management for open source and private projects. Need private repositories?” и ссылка на тарифные планы “Upgraded plans start at $7/mo.”
Справа от слогана находится панель регистрации, со следующими элементами
Поле ввода Тип Подсказка
user[login] text Pick a username
user[email] text Your email
user[password] password Create a password
Ниже находится правило ввода пароля "Use at least one lowercase letter, one numeral, and seven characters.” Кнопка регистрации c текстом "Sign up for GitHub"ниже которой находится текст с ссылками на правила пользования и политику безопастности
